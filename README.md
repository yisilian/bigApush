# 缅A每日推送

A股量化选股 + 每日自动推送系统，通过 GitHub Actions 实现全自动化运行。

## 功能特性

- **智能选股**：多策略并行选股，支持10+种量化策略
- **大盘复盘**：每日推送上证/深证/创业板/科创50指数行情
- **板块热力图**：领涨领跌板块一目了然
- **财经要闻**：自动抓取新浪财经最新资讯
- **新增/去除对比**：每日对比上个交易日选股变化
- **飞书推送**：选股结果 + 复盘分析自动发送到飞书群

## 自动运行时间

| 时间 | 任务 | 内容 |
|------|------|------|
| 15:10（周一-周五） | 收盘选股 | 选股结果 + 大盘复盘 + 板块 + 新闻 |
| 08:00（每天） | 早间新闻 | 大盘回顾 + 板块 + 财经要闻 |
| 22:30（每天） | 晚间新闻 | 收盘行情 + 板块 + 晚间要闻 |

## 快速开始

### 1. Fork 仓库

点击右上角 Fork 按钮，将仓库复制到你的 GitHub 账号下。

### 2. 配置飞书 Webhook

1. 在飞书群中添加自定义机器人
2. 复制 Webhook URL
3. 进入仓库 → Settings → Secrets and variables → Actions
4. 新建 Secret：`FEISHU_WEBHOOK`，值为你的 Webhook URL

### 3. 启用 Actions

进入仓库 → Actions → 点击 "Enable workflow"

### 4. 手动测试

在 Actions 页面 → 缅A每日推送 → Run workflow → 点击绿色按钮运行

## 配置说明

### config/config.yaml

```yaml
# 飞书机器人配置
feishu:
  webhook_url: "https://open.feishu.cn/open-apis/bot/v2/hook/xxxxx"

# 数据更新配置
update:
  lookback_days: 10

# 股票过滤配置
filters:
  min_market_cap: 10      # 最小市值（亿）
  max_market_cap: 10000   # 最大市值（亿）
  exclude_st: true        # 排除ST股票
```

## 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 初始化数据（首次运行）
python main.py init

# 执行选股并推送
python main.py run

# 启动Web界面
python main.py web
```
